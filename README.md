## üß™ How to Use / Reproduce

1. Follow the instructions in `Suricata_Installation_Config.txt` to install and configure Suricata.
2. Place the rules from `local.rules.txt` into your Suricata rule directory (e.g., `/etc/suricata/rules/`).
3. Restart Suricata so that it loads the new rules.
4. Run the attack or trigger workflow documented in `Suricata_IDPS_SETUP_Attacks_Demo.txt`.
5. Review the alerts/logs generated by Suricata to confirm detection of the demonstration attacks.
6. Optionally review `NOT_DEMO_OTHER_RULES.txt` for additional custom rule ideas.
